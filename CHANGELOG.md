# Changelog

All notable changes to the Halcyon Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-14

### Added
- Initial release of Halcyon Theme
- Dark color theme with oceanic twilight palette
- Comprehensive syntax highlighting for:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - And many more languages
- Terminal color support
- Optimized UI colors for sidebar, status bar, and editor chrome
- Special highlighting for:
  - Comments (italic, muted)
  - Strings (pastel green)
  - Functions and classes (bright cyan)
  - Keywords (soft purple)
  - Parameters and attributes (warm yellow)
  - Errors and warnings (coral red)
- Multi-level JSON key coloring
- Enhanced Markdown syntax support

### Color Palette
- Background: `#1D2433` (Deep navy)
- Foreground: `#A2AABC` (Soft gray-blue)
- Accent: `#5CCFE6` (Bright cyan)
- Green: `#BAE67E` (Pastel green)
- Yellow: `#FFD580` (Warm yellow)
- Purple: `#C3A6FF` (Soft purple)
- Red: `#F95972` (Coral red)
